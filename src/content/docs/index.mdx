**Price Compare Agent (PCA)** 是一个桌面应用，像一个 AI 导购——你告诉它想买什么，它去京东和淘宝帮你比价，告诉你哪个平台更划算。

## 它怎么工作

```
你说："找一款 300 以内适合运动的蓝牙耳机"
  ↓
AI 理解你的需求 → 从商品库里筛选 → 跨平台匹配比价 → 给你推荐结果
```

不用学任何语法，说人话就行。如果信息不够（比如只说"买个耳机"），它会追问你预算和用途。

## 技术概览

| 层 | 技术 |
|------|------|
| 桌面框架 | Tauri 2 (Rust) |
| 前端 | React 19 + TypeScript + Ant Design 5 |
| 图表 | ECharts |
| AI 接入 | async-openai + reqwest (OpenAI / Anthropic 双协议) |
| 数据 | 离线 JSON，编译时嵌入，无需数据库 |
