**Price Compare Agent (PCA)** 是一个桌面应用程序，用户用自然语言描述想买的商品，AI 自动跨平台比价并给出推荐。

你可以在 [GitHub Release](https://github.com/Badnuker/price-compare-agent/releases) 下载最新版本。

  <Card title="🔍 自然语言比价">

  </Card>
  <Card title="📊 价格可视化">
    比价表格 + 柱状图，不同平台一目了然
  </Card>

  <Card title="⚡ 实时步骤反馈">

  </Card>
</CardGrid>

## 技术概览

| 层 | 技术 |
|------|------|
| 桌面框架 | Tauri 2 (Rust) |
| 前端 | React 19 + TypeScript + Ant Design 5 |
| 图表 | ECharts |
| AI 接入 | async-openai + reqwest (OpenAI / Anthropic 双协议) |
| 数据 | 离线 JSON，编译时嵌入，无需数据库 |
