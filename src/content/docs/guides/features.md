## 自然语言比价

核心功能。用户用日常语言描述购买需求，AI 自动完成整个比价流程。

**输入示例**：
- "找一款 300 以内适合运动的蓝牙耳机"
- "1000 以下降噪耳机推荐"
- "帮我看看红米 Buds 5 在京东和淘宝哪个便宜"

**处理流程**：

1. **意图解析** — LLM 提取商品类型、品牌、预算、功能需求
2. **商品筛选** — 从离线数据集中粗筛候选商品
3. **比价分析** — LLM 匹配跨平台同款、按价格排序、生成推荐
4. **结果展示** — 表格 + 柱状图 + 推荐文案

## 跨平台匹配

不同电商平台对同一商品命名可能完全不同。系统通过 LLM 语义理解能力自动识别：

| 京东名称 | 淘宝名称 | 匹配结果 |
|---------|---------|---------|
| 小米 Redmi Buds 5 | 红米 Buds5 | 完全匹配 |
| 华为 FreeBuds 5i | 华为 FreeBuds5i | 完全匹配 |
| 漫步者 X3 Air | Edifier 漫步者 X3 Air | 完全匹配 |

匹配类型分为三种：

- **完全匹配** (exact)：同一商品在不同平台
- **相似** (similar)：规格接近但有差异
- **替代推荐** (alternative)：功能满足但型号不同

## 价格可视化

比价结果以两种形式展示：

- **表格**：商品名、平台（彩色标签）、价格（红色加粗）、规格、匹配类型、链接
- **柱状图**：ECharts 绘制的跨平台价格对比图，不同平台不同颜色

表格默认按价格升序排列，支持点击列头手动排序。

## 实时步骤展示

搜索过程中，进度条实时显示 Agent 执行状态：

```
理解需求 → 筛选商品 → 比价分析 → 生成推荐
```

每一步完成后由后端通过 Tauri event 实时推送到前端，精准反映真实进度。

## 双模型兼容

应用内支持两种 AI 调用格式，在设置页一键切换，即时生效，无需重启：

- **OpenAI 兼容格式** — 兼容 OpenAI Chat Completion API 的所有服务商
- **Anthropic 格式** — 兼容 Anthropic Messages API 的所有服务商

## 流式输出

AI 回复不再等待全部完成后一次性显示，而是像打字一样逐字输出。支持 OpenAI 和 Anthropic 双端的 SSE streaming。用户可以实时看到 LLM 的分析过程，体验更流畅。

## 暗色主题

全界面 GitHub Dark 风格暗色主题，通过 CSS 变量体系覆盖所有 Ant Design 组件。长时间使用不疲劳，OLED 屏幕更省电。

## 对话历史

左侧 Sidebar 保存历史对话记录，支持多轮追问。Agent 会携带上下文信息，用户说"把预算改成 500"时，AI 知道之前讨论的是什么商品。

## 思考可见

LLM 的推理过程通过 ThinkingBlock 组件展示，用户可以展开查看 AI 如何分析需求、筛选商品、生成推荐。适合调试 prompt 和了解 AI 决策逻辑。

## 欢迎页

首次打开应用时展示 WelcomeScreen，包含常用的搜索建议卡片，点击即可快速开始比价。

## 智能追问

当用户输入信息不足时，AI 不会盲目推荐，而是追问补充。例如输入"买个耳机"，系统会提示：

> 请补充以下信息以获取更准确的比价结果：品牌、预算范围、使用场景
